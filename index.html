SDK.js

Un simple générateur de sudokus

Qu'est-ce que c'est?

SDK.js permet de générer un sudoku en JavaScript, directement depuis le navigateur.
il est possible de définir une clé : pour une valeur donnée, SDK.js génère toujours le même sudoku. La clé peut être n'importe quelle chaîne de caractères ("Hello!", "Toto32", etc...).
Si vous n'avez pas d'inspiration, une clé aléatoire est utilisée...
